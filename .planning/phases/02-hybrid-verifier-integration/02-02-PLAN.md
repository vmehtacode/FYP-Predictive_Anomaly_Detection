**`HybridVerifierAgent.evaluate(self, forecast, scenario=None, timestamps=None, return_details=False) -> float | tuple[float, dict]`**:
CRITICAL: Signature MUST match exactly: `forecast: np.ndarray, scenario: Optional[ScenarioProposal] = None, timestamps: np.ndarray | None = None, return_details: bool = False`

Implementation flow (see 02-RESEARCH.md Pattern 1):
1. Run physics_layer.evaluate(forecast) -> physics_scores, physics_details
2. Compute early_exit_mask = physics_scores > config.early_exit_threshold
3. If GNN model is available AND graph_data is available: run _run_gnn_inference(forecast, early_exit_mask) -> gnn_scores. Else: gnn_scores = np.zeros_like(physics_scores)
4. If graph_data available: run cascade_layer.evaluate(physics_scores, gnn_scores, graph_data.edge_index) -> cascade_scores. Else: cascade_scores = np.zeros_like(physics_scores)
5. _combine_scores(physics, gnn, cascade, early_exit_mask) -> combined, breakdown
6. _compute_reward(combined, scenario) -> reward
7. If return_details: return (reward, breakdown). Else: return reward

**`_compute_reward(self, combined_scores, scenario) -> float`** (see 02-RESEARCH.md Pattern 5):
- Compute mean anomaly score across all nodes
- Confidence = abs(mean_score - 0.5) * 2 (scaled to [0, 1])
- If scenario present: expect anomaly (mean_score > 0.5 = correct)
- If no scenario: expect normal (mean_score <= 0.5 = correct)
- Correct prediction: reward = +confidence
- Incorrect prediction:
  - False negative (scenario present but score <= 0.5): reward = -confidence * config.false_negative_penalty_ratio
  - False positive (no scenario but score > 0.5): reward = -confidence
- Clip to [-1, +1] range
- Return float
